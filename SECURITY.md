# Security Policy

## 🔐 API Key Safety

QuestCraft never sends your API keys to our servers. All AI requests go directly from your browser to the provider you choose. API keys are stored in your browser's sessionStorage (cleared when you close the tab) or, for the default Gemini key, in your `.env.local` file at build time.

## 🐛 Reporting a Vulnerability

If you find a security issue, **do not** open a public issue. Email `security@cashlessconsumer.in` directly.

We'll respond within 48 hours and coordinate disclosure.
