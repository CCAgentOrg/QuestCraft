**QuestCraft** turns any topic into a playable, AI-driven board game. Build custom quests, learn through play, and generate dynamic scenarios — all powered by your choice of AI model (Gemini, OpenAI, OpenRouter, or community gateways).

Built for classrooms, self-learners, and game enthusiasts.

## ✨ Features

- **🎮 Board Game Engine** — Monopoly-style gameplay with resources, dice, jail, and turn-based mechanics
- **🤖 AI-Powered Scenarios** — Every property landing triggers a unique, context-aware challenge generated by AI
- **🧩 Quest System** — 8 built-in quests (Aadhaar, AI Ethics, Climate Change, Gender Equality, Urban Transit, and more)
- **🔧 Quest Maker** — Visual wizard to create and share your own quests
- **🌍 4 Languages** — English, Spanish, Hindi, Tamil with full i18n
- **🔌 Multi-Provider AI** — BYO API key for Gemini, OpenAI, Groq, Together AI, OpenRouter, or free community tier
- **📊 Usage Analytics** — Token tracking, cost estimation, audit logging
- **📱 Responsive** — Mobile-first design, works on any screen

## 🚀 Quick Start

```bash
# Clone the fork
git clone https://github.com/CCAgentOrg/QuestCraft.git
cd QuestCraft

# Install dependencies
npm install

# Add your Gemini API key to .env.local
echo "GEMINI_API_KEY=your_key_here" > .env.local

# Start dev server
npm run dev
```

Then open `http://localhost:5173` and pick a quest to play.
